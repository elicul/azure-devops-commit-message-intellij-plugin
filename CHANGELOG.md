## [1.0.3] - 2026-04-01

### Changed

- Preserve the existing commit message text and refresh only the Azure DevOps work item prefix.
- Keep manual edits or AI-generated commit summaries intact when the prefix is refreshed.
- Pin the build to Java 11 for IntelliJ Platform 2021.3 compatibility.

## [1.0.2] - 2023-05-09

### Added

- Enclosing Azure DevOps ID with square brackets.


## [1.0.1] - 2023-05-03

### Added

- Refresh icon on Commit dialog added.


## [1.0.0] - 2023-04-30

### Added

- Initial project added.
